# swift-http-client

Swift HTTP Client is a simple HTTP client implementation on top of URLSession using `swift-http-types`. It provides a common set of defaults, focusing on modern Swift features.

## Getting Started

Add the following dependency clause to your Package.swift:

```swift
dependencies: [
  .package(url: "https://github.com/grsouza/swift-http-client.git", from: "0.1.0")
]
```
